# 핵심기후인자 학습 모델 요약

- 학습 목표: 한반도 월평균기온과 강수량의 `-, 0, +` 3분위 기호
- 입력 시차 규칙: 대부분의 지수는 `M-1`, 지연 갱신 지수(예: `ONI`, `QBO`)는 `M-2`까지만 사용
- 파생 입력: 최신 가용값(`m0`), 1개월 전(`m1`), 3개월 평균(`roll3`), 1개월 변화(`delta1`)
- 모델군: 기후평년, 유사사례 kNN, 소프트맥스 회귀, LightGBM
